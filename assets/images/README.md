# Placeholder Images

Voeg hier je eigen afbeeldingen toe voor de feature cards op de homepage.

## Aanbevolen afmetingen:
- Hero/Header image: 1920x600px
- Feature card icons: 300x300px (vierkant)

## Benodigde afbeeldingen:
- `hero-gradient.jpg` - Achtergrond voor de header (of laat leeg voor pure gradient)
- `music-icon.png` - Icoon voor muziek sectie
- `schedule-icon.png` - Icoon voor rooster
- `price-icon.png` - Icoon voor prijzen
- `testimonial-icon.png` - Icoon voor testimonials
- `info-icon.png` - Icoon voor info
- `calendar-icon.png` - Icoon voor social media

## Tijdelijke oplossing:
Je kunt emoji's gebruiken in de feature_row titels (zoals nu) of gratis iconen downloaden van:
- https://www.flaticon.com/
- https://icons8.com/
- https://www.iconfinder.com/

Of gebruik gekleurde vierkanten met emoji's/tekst als placeholder.
